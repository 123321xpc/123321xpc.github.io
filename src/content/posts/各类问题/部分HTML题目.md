# 如何理解语义化？

+ 在特定的情境下使用特定的标签，使得代码可读性更高，让搜索引擎更容易读懂。例如，标题使用`<h1>, <header>`，段落使用`<p>`，列表用`ul li`等


# 那些是内联元素，那些是块状元素？

+ 块状`display: block/table`：`div, h1, table, ul, ol, p`（会自动换行）
+ 内联`display: inline/inline-block`：`span, img, input, button`（不会自动换行）
